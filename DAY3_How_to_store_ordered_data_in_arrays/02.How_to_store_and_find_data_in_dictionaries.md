 How_to stor and find data in dictionaries

 怎麼樣正確管理與儲存array資料

 ``` Swift
 var employee = ["Taylor", "Singer", "Nashville"]

 print("Name: \(employee[0])")
 employee.remove(at: 1)
 print("Job tittle: \(employee[1])")
 print("Location: \(employee[2])")

 // 如果我擅自移除其中的元素, 就會造成代碼錯誤
 ```

